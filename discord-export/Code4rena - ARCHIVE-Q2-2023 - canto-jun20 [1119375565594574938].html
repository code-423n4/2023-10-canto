kt_nowk 06/20/2023 20:40
Sure. If you send a token from Ethereum to Canto using an IBC transfer, the IBC transfer will still be successful in the event that the swap for 4 canto or the automatic ERC20 conversion fails. If you send tokens to Canto using IBC, they will always end up on Canto (never back to the origin chain) (edited)
